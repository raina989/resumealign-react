# 🎯 ResumeAlign - AI Resume & Job Description Matcher

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://resumealign-txfwxaovexerwupdz45r2d.streamlit.app/)

## 📋 Overview

ResumeAlign is an intelligent tool that analyzes your resume against job descriptions using NLP and machine learning. It provides match scores, identifies skill gaps, and generates AI-powered suggestions to optimize your resume.

## ✨ Features

- 📄 **Smart Resume Parsing** - Extract text from PDF, DOCX, and TXT files
- 🎯 **Match Analysis** - Calculate similarity scores using TF-IDF and cosine similarity
- 🔧 **Skill Extraction** - Automatically identify technical and soft skills
- 📊 **Visual Reports** - Progress bars and metrics for easy understanding
- 🚀 **AI Suggestions** - Generate resume bullet points and learning resources
- 📜 **History Tracking** - Save and revisit past analyses
- 📥 **Export Reports** - Download detailed TXT and HTML reports
